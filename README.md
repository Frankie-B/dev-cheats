# Dev Cheats

Technical Cheat Sheets for Devs in Hurry. Dev Cheats is a curated [collection](https://dev-cheats.surge.sh) of handcrafted cheat-sheets for devs. Download beautiful cheat-sheets on topics like git, VS Code in PDF format. More cheat-sheets are coming up soon! All the cheat-sheets are in this [repo](https://github.com/godcrampy/cheat-sheets).
